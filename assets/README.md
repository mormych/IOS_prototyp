# 🚗 Car Manager

## 📄 Opis projektu
Aplikacja **Car Manager** to proste narzędzie mobilne przeznaczone dla kierowców, umożliwiające zarządzanie informacjami o posiadanych pojazdach oraz kontrolę ważnych terminów, takich jak przegląd techniczny czy ubezpieczenie OC.

Celem aplikacji jest ułatwienie użytkownikowi codziennego funkcjonowania poprzez zebranie wszystkich kluczowych informacji o pojazdach w jednym miejscu oraz przypominanie o zbliżających się terminach. Dzięki temu użytkownik może uniknąć nieprzyjemnych sytuacji, takich jak jazda bez ważnego przeglądu lub ubezpieczenia.

Aplikacja została zaprojektowana jako intuicyjna i prosta w obsłudze, z przejrzystym interfejsem umożliwiającym szybkie dodawanie, przeglądanie oraz edytowanie danych pojazdów.
